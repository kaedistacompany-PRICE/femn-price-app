# اپ اطلاع‌رسانی قیمت فرومنگنز و آلیاژها
**ارائه شده توسط شرکت شتاب تجهیز آوران (STA)**

## مهم: همه فایل‌ها باید مستقیم کنار هم در ریشه Repository باشند
این پروژه عمداً بدون پوشه‌های فرعی طراحی شده تا در آپلود اشتباه پیش نیاید. وقتی فایل‌ها را آپلود می‌کنید، همه‌ی این فایل‌ها باید **همزمان و کنار هم** در ریشه‌ی مخزن (Repository) باشند، نه داخل پوشه‌ای دیگر.

## شماره واتساپ و تماس (دیگر فرمت خاصی لازم نیست!)
فایل `app.js` را باز کنید، این دو خط نزدیک بالای فایل را پیدا کنید:

```js
WHATSAPP_NUMBER_RAW: "0912 000 0000",
CALL_NUMBER_RAW: "0912 000 0000",
```

شماره واقعی خودتان را به **هر فرمتی** که راحت‌ترید بنویسید — با صفر اول، با فاصله، با یا بدون +98 — کد خودش آن را اصلاح می‌کند. مثلا همه این‌ها قابل قبولند:
- `"0912 345 6789"`
- `"912 345 6789"`
- `"+98 912 345 6789"`

⚠️ نکته مهم: بعد از تغییر این خطوط، **حتما کاراکترهای گیومه `"` و کاما `,` انتهای خط را دست نزنید** — اگر این‌ها به‌هم بریزند، کل اسکریپت از کار می‌افتد و هیچ دکمه‌ای (نه واتساپ، نه نصب اپ) کار نمی‌کند. اگر بعد از ویرایش، چیزی روی سایت درست نمایش داده نشد، همین خطا محتمل‌ترین دلیل است.

## آپلود روی GitHub Pages (رایگان)
1. اگر مخزن قبلی مشکل ساختاری دارد، بهتر است آن را پاک کرده و دوباره بسازید (Settings → پایین صفحه → Delete this repository)
2. مخزن تازه با همان نام بسازید
3. تمام فایل‌های این پوشه را با هم انتخاب کنید و در باکس "uploading an existing file" گیت‌هاب رها کنید
4. مطمئن شوید فایل‌ها مستقیم در ریشه هستند (نه داخل پوشه)
5. **Commit changes**
6. Settings → Pages → Branch: `main`, پوشه: `/ (root)` → Save
7. بعد از ۱-۲ دقیقه آدرس سایت فعال می‌شود

## آپدیت روزانه قیمت (فعلا دستی - بعدا گوگل شیت اضافه می‌شود)
1. در گیت‌هاب وارد فایل `data.json` شوید → آیکون مداد (Edit)
2. هر محصول یک بلوک `{ }` است؛ داخل هر بلوک، آرایه `prices` چهار ردیف قیمت دارد (بازار ایران / واردات FOB / واردات CFR / کارگوی DDP)
3. مقادیر `price`، `change_percent` و `change_direction` ("up" یا "down") را عوض کنید
4. بالای فایل، `updated_at` را به تاریخ امروز تغییر دهید
5. **Commit changes**

## افزودن محصول جدید
در `data.json`، یک بلوک محصول کامل را کپی کنید، بعد از آخرین بلوک (با یک کاما قبل از آن) Paste کنید، و `id`، `name`، `name_en` و قیمت‌ها را تغییر دهید.

## تغییر عکس/نمودار/ویدیو
در `data.json`، فیلد `image_url` (یا `video_url` با `media_type: "video"`) را به اسم فایلی که در ریشه‌ی Repository آپلود کرده‌اید تغییر دهید.
